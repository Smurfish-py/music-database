# Music Database
Music database adalah program untuk mengelola data musik yang disukai dengan antarmuka sederhana yang ramah pengguna. Dibuat dengan menggunakan bahasa pemrograman Python, waktu pembuatan program ini hanya memakan waktu sekitar satu minggu sebagai bagian dari proses pembelajaran saya secara mandiri. 

## Cara Menggunakan
Clone project ke direktori lokal
```cli
git clone https://github.com/smurfish-py/music-database
```
Pindah ke direktori proyek
```cli
cd music-database
```
Untuk menjalankan program, eksekusi file `maincontent.py`
```python
python maincontent.py
```
